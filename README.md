# Duck Social Network

O aplicație de tip rețea socială dezvoltată în **Java**, axată pe o arhitectură modulară și o interfață grafică intuitivă.

## Funcționalități Principale
* **Gestiune Completă (CRUD):** Administrare utilizatori și relații de prietenie.
* **Interfață Grafică:** Construită cu **JavaFX** și **FXML** pentru o experiență de utilizator modernă.
* **Actualizări Real-time:** Utilizarea design pattern-ului **Observer** pentru sincronizarea datelor.
* **Performanță:** Implementare de **Multithreading** pentru operațiuni asincrone.

## Tehnologii și Concepte
* **Limbaj:** Java
* **Arhitectură:** Layered Architecture (Repository, Service, Controller)
* **Design Patterns:** Observer, Factory
* **Baze de date:** Integrare SQL pentru persistența datelor.
